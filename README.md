# 🌐 Interactive Portfolio Website

> **사용자가 스크롤하며 개발자인 저를 자연스럽게 알아갈 수 있도록 설계한 인터랙티브 프론트엔드 포트폴리오입니다.**

---

# 📌 Project Overview

기존의 정적인 포트폴리오에서 벗어나 스크롤 기반의 인터랙션과 부드러운 화면 전환을 적용하여 사용자 경험(UX)을 향상시키는 것을 목표로 제작하였습니다.

프로젝트를 단순히 나열하는 방식이 아니라, 사용자가 하나의 스토리를 따라가듯 개발자인 저를 소개하고 프로젝트를 자연스럽게 탐색할 수 있도록 설계하였습니다.

HTML, CSS, JavaScript를 기반으로 개발하였으며 GSAP와 ScrollTrigger를 활용하여 인터랙티브한 애니메이션을 구현하였습니다.

---

# 🎨 Design Concept

이번 포트폴리오는 단순히 정보를 나열하는 웹사이트가 아닌,

**'스크롤을 통해 개발자를 소개하는 인터랙티브 포트폴리오'**

라는 컨셉으로 제작하였습니다.

불필요하게 화려한 효과보다는

- 자연스러운 스크롤
- 넓은 여백
- 미니멀한 UI
- 프로젝트 중심 레이아웃

을 통해 콘텐츠에 집중할 수 있도록 설계하였습니다.

# 🔍 Design Research

프로젝트를 제작하기 전 다양한 인터랙티브 웹사이트를 분석하며 레이아웃, 사용자 경험, 애니메이션 흐름을 연구하였습니다.

참고한 사이트

- https://chaimaajimi.com/
- https://valentingassend.com/
- https://codapress.co.uk/books/google-antigravity-pro/

위 사이트들의

- 스토리텔링 방식
- 화면 전환
- 스크롤 인터랙션
- 타이포그래피
- 여백 활용
- 프로젝트 구성 방식

등을 참고하여 제 포트폴리오에 맞게 재해석하여 구현하였습니다.

※ 디자인과 소스코드를 복제하지 않았으며, 사용자 경험(UX)과 인터랙션 아이디어를 참고하여 직접 구현하였습니다.
